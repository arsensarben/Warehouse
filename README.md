# Warehouse Management System 📦

**Warehouse Management System** — це навчальний проєкт з курсу "Об'єктно-орієнтоване програмування", розроблений на мові C# з використанням Windows Forms. Програма призначена для автоматизації обліку товарів на складі, контролю залишків та ведення історії приходів та витрат.

## 🚀 Основні можливості

* **Керування каталогом товарів:** Додавання, редагування та видалення позицій через зручний інтерфейс.
* **Інтелектуальна валідація:** Захист від створення дублікатів товарів та перевірка коректності введення цін (заборона нульової або від'ємної ціни).
* **Документообіг:** Оформлення накладних на прихід та видачу товару з автоматичним оновленням залишків.
* **Історія операцій:** Перегляд усіх рухів по складу з механізмом **Snapshot**, що зберігає стан назв товарів на момент операції (захист від зміни даних у документах заднім числом).
* **Візуальний контроль:** Автоматичне підсвічування рядків з товарами, що мають нульовий залишок (LightCoral), для швидкого реагування.
* **Persistence:** Повне збереження та автоматичне завантаження даних у форматі **JSON**.

## 🏗️ Архітектура проєкту

Проєкт побудований за принципом розділення відповідальності (**Separation of Concerns**):

* **`Models/`**: Опис сутностей (`Product`, `Waybill`, `WaybillItem`). Відповідають лише за структуру даних.
* **`Services/`**: Бізнес-логіка програми (`Warehouse.cs`). Робота зі списками, проведення розрахунків, серіалізація та десеріалізація даних.
* **`UI/`**: Слой інтерфейсу (Windows Forms). Відображення даних користувачу та обробка подій натискання кнопок.
